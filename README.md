# PRO-TRACKER 2026 🏋️‍♂️

![PRO-TRACKER Logo](app/src/main/res/drawable/app_logo.png)

**PRO-TRACKER 2026** is a modern, high-performance fitness tracking application built with **Jetpack Compose**. Designed for athletes who want precision, ease of use, and deep insights into their training progress.

## 🚀 Features

### 📋 Intelligent Workout Management
- **Predefined Exercise Database:** Comprehensive list of exercises for Chest, Back, Legs, Shoulders, Biceps, Triceps, Core, Forearms, and Glutes.
- **Custom Exercises:** Add your own specialized exercises to the database with ease.
- **Dynamic Training Plans:** Create, rename, and manage multiple workout plans tailored to your goals.

### ⚡ Active Session Tracking
- **Live Timer:** Track your workout duration in real-time with pause/resume functionality.
- **Seamless Logging:** Quickly add and edit sets (Weight x Reps) with a tap.
- **Exercise Swapping:** Pivot your workout on the fly. Swap exercises within the same muscle group without losing your session progress.
- **Video Guidance:** Visual placeholders for exercise techniques (integrated for major movements).

### 📊 Progress & Analytics
- **Workout History:** A detailed archive of every session you've completed.
- **Volume Visualization:** Built-in charts to visualize the total volume (kg) moved per session.
- **Average Metrics:** Automatic calculation of average weight and intensity per exercise.
- **Goal Setting:** Define and track your fitness goals directly in your profile.

### 🎨 Modern Experience
- **Jetpack Compose UI:** A smooth, reactive interface built with the latest Android standards.
- **Dynamic Theming:** Full support for **Dark Mode**, Light Mode, and System Default themes.
- **Animated Splash Screen:** A premium feel from the moment you launch the app.

## 🛠 Tech Stack

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Material 3)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Asynchronous Flow:** Kotlin Coroutines
- **State Management:** Compose States & SnapshotStateLists

## 🛠 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/XER00xx/training-manager-android.git
   ```
2. Open the project in **Android Studio (Ladybug or newer)**.
3. Build and run on your device or emulator.
